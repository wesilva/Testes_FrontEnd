## Introdução aos testes no Javascript

### `npm init -y`

### `yarn add -D jest @types/jest`

### `npm i -g httpserver`

### `yarn add --dev babel-jest @babel/core @babel/preset-env`

### `yarn add lodash`

### `yarn add -D prettier`

### Para atualizar o snapshot digitar 'u' no console do yarn test:watch

### `yarn add -D ndb`
